# ملخص الدرس: Laravel API Resources

## الفكرة الأساسية
API Resource هو طبقة تنظم شكل البيانات قبل إرسالها إلى الواجهة. بدلاً من إرسال Model كامل، نحدد الحقول التي تحتاجها الصفحة فقط.

## لماذا هذا مهم؟
إرسال كل البيانات قد يكشف معلومات غير لازمة ويجعل الواجهة مرتبكة. Resource يجعل البيانات نظيفة ومتوقعة.

## داخل FlowPilot
FlowPilot سيعرض بيانات المشاريع والمهام بشكل منظم وثابت في الواجهة.

## المثال
return [
    "id" => $this->id,
    "name" => $this->name,
    "tasks_count" => $this->tasks_count,
    "created_at" => $this->created_at->format("Y-m-d"),
];

## نتيجة الدرس
طبقة موارد تنظّم البيانات قبل عرضها.
