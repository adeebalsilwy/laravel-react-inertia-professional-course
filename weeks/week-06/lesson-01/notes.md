# ملاحظات الدرس 21: Laravel API Resources وتنظيم البيانات

- المفهوم الأساسي: API Resource طبقة تنظّم شكل البيانات قبل إرسالها للواجهة. بدلاً من إرسال كل حقول Model، يختار المطور ما تحتاجه الصفحة فقط.
- أين يظهر في FlowPilot: سنستخدم ProjectResource لإرسال اسم المشروع وحالته وعدد المهام دون كشف تفاصيل داخلية.
- أهم ملف أو مكان تنفيذ: app/Http/Resources/ProjectResource.php
- راجع الأخطاء الشائعة بعد التطبيق.
