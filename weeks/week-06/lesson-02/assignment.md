# واجب الدرس 22: البحث والفلاتر المتقدمة

## الهدف من الواجب
بناء نظام بحث وفلاتر متكامل لصفحة المهام في FlowPilot باستخدام when() و appends().

## المهام المطلوبة (6 مهام)
1. **البحث الأساسي**: use `when()` + `LIKE` للبحث في عنوان المهمة.
2. **فلتر الحالة**: use `when()` + `where('status', $status)`.
3. **فلتر الأولوية**: use `when()` + `where('priority', $priority)`.
4. **بحث موسع**: use `orWhere()` للبحث في العنوان والوصف معاً.
5. **حفظ الفلاتر**: use `appends()` وتمرير `filters` إلى Inertia.
6. **اختبار**: تصفح الصفحات وتأكد من بقاء الفلاتر.

## معايير القبول
- كتابة كلمة في البحث تظهر نتائج مطابقة
- اختيار حالة أو أولوية يفلتر النتائج
- روابط الترقيم تحتفظ بـ ?search=...&status=...
- صفحة React تعرض الفلاتر الحالية في الـ inputs
