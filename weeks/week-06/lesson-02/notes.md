# ملخص الدرس: البحث والفلاتر المتقدمة

## الفكرة الأساسية
البحث يسمح للمستخدم بالوصول السريع إلى المشروع أو المهمة. الفلاتر تضيق النتائج حسب الحالة أو الأولوية أو المسؤول.

## لماذا هذا مهم؟
عندما يكبر FlowPilot لن يكون من العملي تصفح كل المهام يدوياً. البحث والفلاتر تجعل التطبيق عملياً للاستخدام اليومي.

## داخل FlowPilot
صفحة المشاريع والمهام ستصبح قابلة للبحث والتصفية بطريقة احترافية.

## المثال
$tasks = Task::query()
    ->when(request("search"), fn($q, $search) =>
        $q->where("title", "like", "%{$search}%")
    )
    ->when(request("status"), fn($q, $status) =>
        $q->where("status", $status)
    )
    ->paginate(12)
    ->withQueryString();

## نتيجة الدرس
بحث وفلاتر عملية في صفحات FlowPilot.
