# ملخص الدرس: صفحات Dashboard وWorkspace

## الفكرة الأساسية
Dashboard هي الصفحة التي تعطي المستخدم نظرة سريعة على حالة العمل. Workspace هو المساحة التي تجمع المشاريع والفرق والمهام الخاصة بالمستخدم.

## لماذا هذا مهم؟
المستخدم لا يريد البحث في كل مكان لمعرفة ما يحدث. لوحة التحكم تلخص له الأرقام المهمة وآخر الأنشطة والمهام القريبة.

## داخل FlowPilot
هذه أول شاشة يشعر فيها الطالب أن FlowPilot بدأ يتحول إلى منتج حقيقي وليس مجرد أمثلة منفصلة.

## المثال
return Inertia::render('Dashboard', [
    'stats' => [
        'projects' => 4,
        'openTasks' => 18,
        'doneTasks' => 9
    ]
]);

## نتيجة الدرس
صفحة Dashboard أولية تعرض مؤشرات FlowPilot.
