# 📄 واجب الدرس 5: فهم المسارات في Laravel وربطها بصفحات React

## 🎯 الهدف
بناء مسار (Route) كامل في مشروع FlowPilot يربط رابط `/about-us` بصفحة React تعرض معلومات عن التطبيق.

## 📋 المطلوب بالتفصيل

### المهمة 1: إنشاء مسار جديد (3 نقاط)
- افتح ملف `routes/web.php`
- أضف مساراً من نوع GET للرابط `/about-us`
- استخدم `AboutController` والدالة `index`
- سمِّ المسار باسم `about.index`

### المهمة 2: إنشاء Controller (3 نقاط)
- شغّل الأمر `php artisan make:controller AboutController`
- أضف دالة `index()` تعيد استخدام `Inertia::render()`
- مرر البيانات التالية إلى صفحة React:
  - `appName` ← "FlowPilot"
  - `version` ← "1.0.0"
  - `team` ← مصفوفة بأسماء ["أحمد", "سارة", "خالد"]

### المهمة 3: إنشاء صفحة React (4 نقاط)
- أنشئ ملف `resources/js/Pages/AboutUs.jsx`
- اعرض عنوان "من نحن" في أعلى الصفحة
- اعرض اسم التطبيق والإصدار
- اعرض قائمة بأعضاء الفريق باستخدام `map()`

## ✅ معايير القبول (التقييم)
- الرابط `http://localhost:8000/about-us` يعمل بدون أخطاء (3 نقاط)
- الصفحة تعرض اسم التطبيق والإصدار (2 نقطة)
- قائمة الفريق تظهر بشكل صحيح (2 نقطة)
- الكود منظم ويستخدم Named Route (2 نقطة)
- لا يوجد أخطاء في Console المتصفح (1 نقطة)

المجموع: **10 نقاط**

## ⭐ التحديات الإضافية (للطلاب المتقدمين)
1. أضف مساراً جديداً `/contact` مع Controller خاص به
2. استخدم `Route::redirect('/about', '/about-us')` لإعادة التوجيه
3. أضف صفحة `Projects/Show.jsx` مع Route Parameter (`/projects/{id}`)

## 📬 طريقة التسليم
1. لقطة شاشة للصفحة النهائية في المتصفح
2. نسخ محتوى الملفات المعدلة: `web.php`، `AboutController.php`، `AboutUs.jsx`
3. إرسال كل ما سبق إلى المدرب للمراجعة
