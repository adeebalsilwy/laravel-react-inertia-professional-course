📄 واجب الدرس 5: بناء المسارات الذكية

"المسار هو الباب الذي يدخل منه المستخدم. اليوم ستبني أول باب لمشروع FlowPilot بنفسك!"

🎯 الهدف من الواجب

بعد إتمام هذا الواجب، ستكون قادراً على:

  • إنشاء Route جديد في Laravel وربطه بصفحة React.
  • استخدام Controller لتمرير البيانات إلى الواجهة.
  • استخدام Named Routes لإنشاء روابط ذكية.
  • فهم دورة حياة الطلب من المتصفح إلى الصفحة النهائية.

📋 خطوات الواجب (نفذ بالترتيب)

1

إنشاء مسار "من نحن"

افتح ملف routes/web.php وأضف مساراً جديداً باسم /about-us.

Route::get('/about-us', [AboutController::class, 'index'])   ->name('about.index');

تلميح: لا تنس إضافة use في أعلى الملف لاستيراد Controller.

2

إنشاء Controller

شغّل الأمر التالي في الطرفية:

php artisan make:controller AboutController

تأكد من ظهور الملف في app/Http/Controllers/AboutController.php.

3

كتابة دالة index في Controller

افتح AboutController.php وأضف دالة index() ترجع Inertia::render('AboutUs') مع بيانات:

public function index() {   $data = [     'appName' => 'FlowPilot',     'version' => '1.0.0',     'team' => ['أحمد', 'سارة', 'خالد'],   ];   return Inertia::render('AboutUs', [     'data' => $data   ]); }
4

إنشاء صفحة React

أنشئ ملفاً جديداً باسم resources/js/Pages/AboutUs.jsx:

import { Head } from '@inertiajs/react'; export default function AboutUs({ data }) {   return (     <>       <Head title="من نحن" />       <div className="max-w-2xl mx-auto p-8">         <h1 className="text-4xl font-black mb-4">من نحن</h1>         <p className="text-lg mb-2">اسم التطبيق: {data.appName}</p>         <p className="text-lg mb-4">الإصدار: {data.version}</p>         <h2 className="text-2xl font-bold mb-2">فريق العمل:</h2>         <ul className="list-disc pr-6">           {data.team.map((name, i) => (             <li key={i}>{name}</li>           ))}         </ul>       </div>     </>   ); }
5

اختبار المسار

شغّل السيرفر وافتح المتصفح على الرابط http://localhost:8000/about-us.

php artisan serve

يجب أن ترى صفحة "من نحن" مع اسم التطبيق، الإصدار، وأعضاء الفريق.

✅ معايير القبول (كيف تعرف أنك أنجزت الواجب؟)

  • الوصول إلى /about-us يعرض الصفحة المطلوبة بدون أخطاء.
  • الصفحة تعرض اسم التطبيق "FlowPilot" والإصدار "1.0.0".
  • قائمة الفريق تظهر (أحمد، سارة، خالد) بشكل صحيح.
  • لا يوجد أخطاء 404 أو 500 في المتصفح.
  • ملف Controller نظيف ومنظم (دالة واحدة فقط).

⭐ تحديات إضافية (للطلاب المتقدمين)

  • أضف مساراً ثانياً باسم /contact يعرض صفحة "اتصل بنا" مع بيانات وهمية (بريد إلكتروني، هاتف).
  • استخدم Route::redirect('/about', '/about-us') لتحويل الزوار من رابط /about القديم إلى الجديد.
  • أضف صفحة Projects/Show.jsx تعرض مشروعاً واحداً بالتفصيل باستخدام Route Parameter (/projects/{id}). تلميح: استخدم Route::get('/projects/{id}', ...) و Inertia::render('Projects/Show').

📬 طريقة التسليم

  1. التقط لقطة شاشة (Screenshot) للصفحة النهائية في المتصفح.
  2. انسخ محتوى الملفات التالية في مستند نصي أو تعليق:
    • محتوى routes/web.php (السطور التي أضفتها فقط).
    • محتوى AboutController.php كاملاً.
    • محتوى AboutUs.jsx كاملاً.
  3. أرسل الصورة والملفات إلى المدرب للمراجعة.
