📄 واجب الدرس 7: بناء مكونات React القابلة لإعادة الاستخدام

"قسّم تطبيقك إلى قطع صغيرة، واجعل كل قطعة تتقن وظيفة واحدة."

🎯 الهدف

  • إنشاء 3 مكونات React قابلة لإعادة الاستخدام.
  • تمرير Props بين المكونات.
  • استخدام map() لعرض قائمة بيانات.
  • استخدام العرض الشرطي.

📋 خطوات الواجب

1

إنشاء مجلد Components

أنشئ مجلد resources/js/Components/ (إذا لم يكن موجوداً).

2

مكون UserCard.jsx

أنشئ مكوناً يستقبل Props: name, role, avatar. يعرض بطاقة عضو فريق.

export default function UserCard({ name, role, avatar }) {   return (     <div className="flex items-center gap-4 p-4 border rounded-xl">       <div className="w-12 h-12 rounded-full bg-indigo-100 flex items-center justify-center text-xl font-bold">{avatar}</div>       <div>         <h3 className="font-bold">{name}</h3>         <p className="text-sm text-slate-500">{role}</p>       </div>     </div>   ); }
3

مكون EmptyState.jsx

مكون يعرض رسالة "لا توجد بيانات" مع أيقونة ونص. يستقبل Props: icon, title, message.

4

مكون Badge.jsx

مكون شارة (Badge) يستقبل text و variant (success, warning, danger, info). يظهر بنص ولون حسب الـ variant.

5

استخدام المكونات في Dashboard

افتح Dashboard.jsx واستخدم المكونات الجديدة:

  • استخدم UserCard لعرض 3 أعضاء فريق (بيانات وهمية).
  • استخدم Badge لعرض حالات المشاريع.
  • استخدم EmptyState مع شرط: إذا كانت المصفوفة فارغة.

✅ معايير القبول

  • المكونات الثلاثة تعمل وتعرض البيانات بشكل صحيح.
  • كل مكون يستقبل Props ويعرضها.
  • استخدام map() لعرض قائمة من المكونات.
  • العرض الشرطي يعمل عند وجود بيانات فارغة.
  • الكود منظم - كل مكون في ملف منفصل.

⭐ تحديات إضافية

  • أنشئ مكون ProgressBar.jsx يستقبل percentage ويعرض شريط تقدم.
  • أضف زر "إضافة مشروع" كـ مكون Button.jsx بأنماط مختلفة.
  • استخدم مكون Badge داخل ProjectCard بدلاً من النص العادي.

📬 طريقة التسليم

  1. لقطة شاشة للصفحة تظهر جميع المكونات.
  2. محتوى الملفات: UserCard.jsx, Badge.jsx, EmptyState.jsx, Dashboard.jsx.
  3. إرسال كل ما سبق للمدرب.
