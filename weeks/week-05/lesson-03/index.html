Inertia Form Helper: كيف نرسل البيانات بذكاء؟

"توقف عن القلق بشأن التعامل اليدوي مع النماذج؛ لنتعلم كيف تجعل Inertia تقوم بالمهمة الصعبة عنك."

لماذا نستخدم useForm؟

عندما يملأ المستخدم نموذجاً (Form)، نحتاج لمتابعة ما يكتبه، إرسال البيانات للسيرفر، وإظهار رسائل خطأ إذا نسي حقلاً معيناً. القيام بكل هذا يدوياً في React أمر مرهق.

Inertia توفر لنا أداة تسمى useForm. هي "مساعد ذكي" يتكفل بمتابعة البيانات، حالة الإرسال، وحالات الخطأ تلقائياً.

إدارة رسائل الخطأ

إذا حاول المستخدم حفظ مهمة بدون عنوان، سيقوم Laravel بإعادة "رسالة خطأ". بفضل useForm، ستظهر هذه الرسالة في واجهة React فوراً دون أن نضطر لكتابة أي كود جلب معقد.

مثال لتقريب الفكرة (المترجم الشخصي)

تخيل useForm كأنه "سكرتير" يقف بجانبك وأنت تملأ استمارة.

- هو يسجل كل ما تكتبه فوراً.
- عندما تنتهي، يأخذ الاستمارة ويجري بها للمكتب المختص (Laravel).
- إذا كان هناك نقص، يعود لك ويؤشر بالقلم الأحمر على المكان الذي أخطأت فيه!

التطبيق العملي

الخطوة 1: استخدام useForm في المكون

import { useForm } from '@inertiajs/react';

const { data, setData, post, errors, processing } = useForm({

  name: '',

});

الخطوة 2: عرض رسالة الخطأ

{errors.name && <div className="text-red-500">{errors.name}</div>}
