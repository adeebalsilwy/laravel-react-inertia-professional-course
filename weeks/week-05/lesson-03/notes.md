# ملخص الدرس: Inertia Forms وحالات التحميل

## الفكرة الأساسية
useForm أداة من Inertia تساعد React على إرسال النماذج إلى Laravel ومتابعة الأخطاء وحالة التحميل بسهولة.

## لماذا هذا مهم؟
النماذج جزء أساسي من FlowPilot: إنشاء مشروع، إضافة مهمة، تعديل ملف شخصي. إذا لم نتعامل مع الأخطاء والتحميل ستصبح تجربة المستخدم ضعيفة.

## داخل FlowPilot
المستخدم سيضيف مشاريع ومهام بثقة لأنه يرى الأخطاء فوراً ويعرف أن الطلب قيد المعالجة.

## المثال
const form = useForm({ title: "", description: "" });

function submit(e) {
  e.preventDefault();
  form.post(route("tasks.store"));
}

## نتيجة الدرس
نماذج تفاعلية مع أخطاء وتحميل ورسائل نجاح.
