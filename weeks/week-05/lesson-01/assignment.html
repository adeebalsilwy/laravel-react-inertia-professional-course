واجب الدرس 17: إنشاء المشاريع وإدارتها CRUD

"سنبني اليوم أول واجهة تسمح للمستخدم بحفظ بيانات حقيقية في قاعدة البيانات والعودة لرؤيتها وتعديلها وحذفها."

الهدف من الواجب

بناء ProjectController متكامل بـ 7 دوال (index, create, store, show, edit, update, destroy) مع التحقق من البيانات ورسائل النجاح. ربط كل ذلك مع واجهة React عبر Inertia.js. بعد هذا الواجب، سيكون لديك نظام إدارة مشاريع كامل يعمل في FlowPilot.

خطوات الواجب (10 خطوات)

الخطوة 1: إنشاء Model مع Migration

في terminal، شغّل الأمر التالي:

php artisan make:model Project -m

تأكد من ظهور ملفين: app/Models/Project.php وملف migration في database/migrations/.

الخطوة 2: تعديل ملف Migration

افتح ملف migration وأضف الأعمدة داخل دالة up():

Schema::create('projects', function (Blueprint $table) {
  $table->id();
  $table->string('name', 255);
  $table->text('description')->nullable();
  $table->foreignId('user_id')->constrained()->cascadeOnDelete();
  $table->timestamps();
});

ثم شغّل php artisan migrate لإنشاء الجدول.

الخطوة 3: إضافة $fillable في Project Model

في app/Models/Project.php، أضف الخاصية:

protected $fillable = ['name', 'description', 'user_id'];

بدون هذا السطر، ستحصل على خطأ MassAssignmentException عند الحفظ.

الخطوة 4: إنشاء ProjectController

شغّل الأمر (الـ --resource تنشئ 7 دوال تلقائياً):

php artisan make:controller ProjectController --resource

تأكد من ظهور الملف في app/Http/Controllers/ProjectController.php.

الخطوة 5: كتابة دالة index()

عدّل دالة index() في Controller:

public function index()
{
  $projects = Project::all();
  return Inertia::render('Projects/Index', [
    'projects' => $projects,
    'success' => session('success'),
  ]);
}

تأكد من إضافة use App\Models\Project; و use Inertia\Inertia; أعلى الملف.

الخطوة 6: كتابة store() مع Validation

أضف التحقق من البيانات في دالة store():

public function store(Request $request)
{
  $validated = $request->validate([
    'name' => 'required|string|min:3|max:255',
    'description' => 'nullable|string|max:1000',
  ]);
  $validated['user_id'] = auth()->id();
  Project::create($validated);
  return redirect()->route('projects.index')->with('success', 'تم إنشاء المشروع');
}

جرب إرسال نموذج باسم فارغ وسترى رسالة الخطأ تعود تلقائياً.

الخطوة 7: كتابة update() و destroy()

دوال التعديل والحذف:

public function update(Request $request, Project $project)
{
  $validated = $request->validate(['name' => 'required|min:3']);
  $project->update($validated);
  return redirect()->route('projects.index')->with('success', 'تم التعديل');
}

public function destroy(Project $project)
{
  $project->delete();
  return redirect()->route('projects.index')->with('success', 'تم الحذف');
}

الخطوة 8: تسجيل المسارات (Routes)

في routes/web.php، أضف:

use App\Http\Controllers\ProjectController;

Route::middleware(['auth'])->group(function () {
  Route::resource('projects', ProjectController::class);
});

وضعنا المسارات داخل auth middleware لضمان أن المستخدم مسجل الدخول أولاً. شغّل php artisan route:list لتأكيد الـ 7 مسارات.

الخطوة 9: إنشاء صفحات React

أنشئ 3 مكونات React في resources/js/Pages/Projects/:

resources/js/Pages/Projects/Index.jsx ← عرض القائمة
resources/js/Pages/Projects/Create.jsx ← نموذج الإنشاء
resources/js/Pages/Projects/Edit.jsx ← نموذج التعديل

في كل ملف، استخدم usePage().props لاستقبال البيانات من Laravel. استخدم useForm من @inertiajs/react للنماذج.

الخطوة 10: الاختبار النهائي

شغّل الخادم واختبر كل عملية:

php artisan serve
  • ✅ زر "إضافة مشروع" → يفتح النموذج → يحفظ → يرجع للقائمة.
  • ✅ القائمة تظهر كل المشاريع المسجلة.
  • ✅ زر تعديل → يغير البيانات → يرجع برسالة نجاح.
  • ✅ زر حذف → تأكيد → يحذف → يرجع برسالة نجاح.
  • ✅ إذا تركت الاسم فارغاً → يظهر خطأ أحمر.

تحدي إضافي (للنجوم ⭐)

أضف ميزة "نسخ المشروع" (Duplicate):

  • 1. أضف زر "نسخ" بجانب كل مشروع في القائمة.
  • 2. أنشئ مسار جديد: Route::post('projects/{project}/duplicate', [ProjectController::class, 'duplicate']).
  • 3. في دالة duplicate()، استخدم replicate() في Eloquent لنسخ المشروع:
public function duplicate(Project $project)
{
  $newProject = $project->replicate();
  $newProject->name = $project->name . ' (نسخة)';
  $newProject->save();
  return redirect()->route('projects.index')->with('success', 'تم نسخ المشروع');
}
