واجب: تحسين استعلامات FlowPilot

"التطبيق البطيء يخسر المستخدمين. اجعل استعلاماتك سريعة."

نظرة عامة

في هذا الواجب، ستطبق تقنيات تحسين الاستعلامات على مشروع FlowPilot: Eager Loading، Query Scopes، ::when()، Chunking، Caching، select().

⏱ الوقت: 90-120 دقيقة 💻 كود + اختبار أداء ⭐ 20% من تقييم الأسبوع

الخطوات

الخطوة 1: تحسين Project Model بـ Query Scopes

أضف هذه الـ Scopes في Project Model:

scopeActive($query) → where('status', 'active')
scopeStatus($query, $status) → where('status', $status)
scopeWithEssential($query) → with(['user:id,name', 'tasks'])+withCount

الخطوة 2: إنشاء Controller محسّن

$ php artisan make:controller ProjectController

اكتب دالة index() تستخدم ::when() للبحث والفلترة:

$projects = Project::select('id','name','status','deadline')
  ->withEssential()
  ->when(request('status'), fn($q,$v) => $q->status($v))
  ->when(request('search'), fn($q,$v) => $q->where('name','like',"%$v%"))
  ->latest()->paginate(10);

الخطوة 3: إضافة Caching

لفّ الاستعلام بـ Cache::remember('projects.index', 3600, function() { ... }).

الخطوة 4: مسح الـ Cache تلقائياً

في Project Model booted()، أضف:

static::created(fn() => Cache::forget('projects.index'));
static::updated(fn() => Cache::forget('projects.index'));
static::deleted(fn() => Cache::forget('projects.index'));

الخطوة 5: اختبار الأداء قبل وبعد

في Tinker، استخدم DB::enableQueryLog() لقياس عدد الاستعلامات:

// قبل التحسين:
DB::enableQueryLog();
$projects = Project::all();
foreach($projects as $p) { echo $p->user->name; }
dump(DB::getQueryLog()); // 1 + عدد المشاريع استعلام!

// بعد التحسين:
DB::enableQueryLog();
$projects = Project::select('id','name')->with('user:id,name')->get();
foreach($projects as $p) { echo $p->user->name; }
dump(DB::getQueryLog()); // استعلامان فقط!

الخطوة 6: استخدام Chunking

اكتب أمر Artisan php artisan make:command ArchiveOldProjects يستخدم chunkById لأرشفة المشاريع القديمة.

⭐ التحدي الإضافي

  1. أضف Scope اسمه scopeForUser($query, $userId) يجلب مشاريع مستخدم معين.
  2. استخدم withExists() لإضافة عمود has_tasks (boolean) يحدد إن كان للمشروع مهام.
  3. أنشئ Middleware يوضح وقت تنفيذ كل استعلام في أسفل الصفحة (Debug Bar بسيط).
  4. استخدم Lazy Collections: Project::cursor() بدلاً من chunk للمقارنة.
