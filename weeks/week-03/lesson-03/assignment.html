واجب: إنشاء Eloquent Models مع العلاقات لـ FlowPilot

"النماذج (Models) هي لغة التخاطب بين PHP وقاعدة البيانات. أتقنها تتقن Laravel."

نظرة عامة

في هذا الواجب، ستقوم بإنشاء Models لـ User و Project و Task مع جميع العلاقات. ستكتب الكود بنفسك وتختبر الاستعلامات في tinker.

⏱ الوقت: 60-90 دقيقة 💻 كود PHP + Tinker ⭐ 20% من تقييم الأسبوع

الخطوات

الخطوة 1: إنشاء Models

$ php artisan make:model Project -m
$ php artisan make:model Task -m
$ php artisan make:model Team -m

لاحظ الـ User Model موجود مسبقاً في Laravel. لا تنشئه.

الخطوة 2: إضافة العلاقات إلى User Model الموجود

افتح app/Models/User.php وأضف:

public function ownedProjects() { return $this->hasMany(Project::class, 'user_id'); }
public function memberOfProjects() { return $this->belongsToMany(Project::class)->withPivot('role', 'joined_at')->withTimestamps(); }
public function assignedTasks() { return $this->hasMany(Task::class, 'assigned_to'); }

الخطوة 3: كتابة Project Model كاملاً

أضف protected $fillable و العلاقات التالية: user (belongsTo)، tasks (hasMany)، members (belongsToMany).

الخطوة 4: كتابة Task Model كاملاً

أضف protected $fillable و العلاقات: project (belongsTo)، assignee (belongsTo مع User::class, 'assigned_to').

الخطوة 5: اختبار العلاقات في Tinker

$ php artisan tinker

جرب هذه الأوامر داخل tinker:

$user = User::first();
$user->ownedProjects; // مشاريعه
$project = Project::first();
$project->user; // صاحب المشروع
$project->tasks; // مهام المشروع
$project->members; // أعضاء المشروع
$project->load('tasks.assignee'); // مهام مع المسؤول

الخطوة 6: إضافة Accessor للـ status

في Project Model، أضف Accessor يحول status إلى العربية.

الخطوة 7: اختبار N+1 وحلها

// قبل الحل (N+1):
foreach (Project::all() as $p) { echo $p->user->name; }

// بعد الحل (2 استعلامات):
foreach (Project::with('user')->get() as $p) { echo $p->user->name; }

استخدم DB::enableQueryLog() في tinker لمشاهدة عدد الاستعلامات.

⭐ التحدي الإضافي

  1. أضف Accessor في Task Model يحوّل priority إلى العربية (low=منخفضة, medium=متوسطة, high=عالية, critical=حرجة).
  2. أضف Mutator في Project Model يجعل أول حرف من اسم المشروع كبيراً (title case).
  3. أضف علاقة team في Project Model (كل مشروع ينتمي لفريق - belongsTo).
  4. أنشئ Query Scope في Project Model لتصفية المشاريع النشطة: Project::active()->get().
