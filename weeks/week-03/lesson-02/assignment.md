# واجب الدرس 10: Migrations وSeeders وFactories

## الهدف
تطبيق الدرس بنفسك داخل مشروع FlowPilot بدون نسخ أعمى.

## المطلوب خطوة بخطوة
1. أنشئ migration لجدول projects
2. أضف أعمدة name وdescription وuser_id
3. شغل migrate
4. أنشئ seeder يضيف مشروعاً تجريبياً

## معايير القبول
- المشروع يعمل بدون أخطاء واضحة.
- تستطيع شرح الملفات التي عدلتها.
- يوجد Commit باسم واضح لهذا الدرس.
- النتيجة ظاهرة في المتصفح أو موثقة بصورة/ملاحظة.
