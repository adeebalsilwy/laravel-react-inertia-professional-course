# ملاحظات الدرس 10: Migrations وSeeders وFactories للمبتدئين

- المفهوم الأساسي: Migration هو ملف يصف شكل جدول في قاعدة البيانات. Seeder يضيف بيانات تجريبية. Factory يصنع بيانات وهمية منظمة. هذه الأدوات تجعل بناء قاعدة البيانات قابلاً للتكرار.
- أين يظهر في FlowPilot: سننشئ جداول المشاريع والمهام ونضيف بيانات تجريبية ليشاهد الطالب FlowPilot مليئاً بالمحتوى أثناء التطوير.
- أهم ملف أو مكان تنفيذ: Terminal
- راجع الأخطاء الشائعة بعد التطبيق.
