📝 كويز الدرس 13: المصادقة (Authentication)

"اختبر فهمك لكيفية حماية تطبيق FlowPilot بنظام الدخول."

عدد الأسئلة: 9 (5 اختيار من متعدد + 2 صح/خطأ + 1 نظري + 1 عملي)

1. (اختيار من متعدد) ما هو تعريف Authentication في تطبيقات الويب؟

✅ الشرح: Authentication هي التحقق من الهوية (من أنت؟). تحديد الصلاحيات هو Authorization (الدرس القادم).

2. (اختيار من متعدد) ما هي وظيفة Laravel Breeze؟

✅ الشرح: Breeze هو Starter Kit جاهز يتضمن كل صفحات المصادقة مع React + Tailwind CSS.

3. (اختيار من متعدد) أي Middleware تستخدم لحماية مسار من الزوار غير المسجلين؟

✅ الشرح: middleware(['auth']) تمنع الزوار غير المسجلين. guest تسمح فقط للزوار غير المسجلين.

4. (اختيار من متعدد) كيف يمكننا الوصول لبيانات المستخدم المسجل في React مع Inertia؟

✅ الشرح: Inertia يشارك بيانات المستخدم عبر HandleInertiaRequests ويجعلها متاحة عبر usePage().props.auth.user.

5. (اختيار من متعدد) ماذا يحدث عندما يحاول زائر غير مسجل فتح /dashboard المحمي بـ auth؟

✅ الشرح: Middleware 'auth' يتحقق من Session. إذا لم يكن المستخدم مسجلاً، يقوم بعمل Redirect إلى route('login').

6. (صح/خطأ) المصادقة (Authentication) هي نفسها التفويض (Authorization).

✅ الشرح: العبارة خطأ. Authentication تثبت الهوية، Authorization تحدد الصلاحيات. هما مفهومان مختلفان.

7. (صح/خطأ) Laravel Breeze يشتغل فقط مع Vue.js ولا يدعم React.

✅ الشرح: خطأ. Breeze يدعم React و Vue.js معاً. الأمر php artisan breeze:install react يثبته مع React.

8. (سؤال نظري) اشرح بجملتين: لماذا نستخدم Laravel Breeze بدلاً من كتابة نظام المصادقة من الصفر؟

💡 الإجابة النموذجية: نستخدم Laravel Breeze لأنه يوفر نظام مصادقة آمن وجاهز مبني من قبل خبراء أمان، مما يوفر وقت التطوير ويضمن اتباع أفضل الممارسات (Best Practices) في حماية البيانات. بدلاً من كتابة آلاف الأسطر من الكود واختبارها، نحصل على حل متكامل بفضل واحد.

9. (سؤال عملي) أكتب كود React (باستخدام Inertia) يعرض اسم المستخدم المسجل إذا كان موجوداً، أو زر "تسجيل الدخول" إذا كان الزائر غير مسجل.

💡 الإجابة النموذجية:
import { usePage } from '@inertiajs/react';

export default function Navbar() {
const { props } = usePage();
const user = props.auth.user;

return (
<div>
{user ? (
<p>مرحباً {user.name}</p>
) : (
<a href="#">تسجيل الدخول</a>
)}
</div>
);
}
