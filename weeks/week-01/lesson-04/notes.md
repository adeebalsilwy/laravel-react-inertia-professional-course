# ملاحظات الدرس 4: Git وتنظيم المستودع وطريقة حفظ التقدم

- المفهوم الأساسي: Git هو نظام يحفظ تاريخ المشروع. كل تغيير مهم يمكن تسجيله في نقطة اسمها commit. عندما يخطئ الطالب يستطيع الرجوع إلى نسخة سابقة بدلاً من فقدان العمل.
- أين يظهر في FlowPilot: في FlowPilot سنحفظ نهاية كل درس في commit واضح، مثل lesson-05-routing-projects، حتى يصبح تقدم الطالب مرتباً.
- أهم ملف أو مكان تنفيذ: Terminal
- راجع الأخطاء الشائعة بعد التطبيق.
