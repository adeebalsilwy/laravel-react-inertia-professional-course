# ملاحظات الدرس 4: Git وتنظيم المستودع والمعايير

## ملخص سريع
نتعلم حفظ التغييرات بطريقة آمنة ومنظمة حتى يعرف المبتدئ كيف يتابع تقدمه.

## أهم المصطلحات
- ما هو Git ولماذا نستخدمه
- معنى commit وbranch
- كتابة رسائل Commit مفهومة
- تنظيم أسماء الملفات والمجلدات

## قاعدة ذهبية للمبتدئ
لا تحفظ الكود قبل أن تعرف أين يكتب ولماذا يكتب وما النتيجة المتوقعة منه.
