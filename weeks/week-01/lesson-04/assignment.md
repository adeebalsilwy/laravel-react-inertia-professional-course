# واجب الدرس 4: Git وتنظيم المستودع والمعايير

## الهدف
تطبيق الدرس بنفسك داخل مشروع FlowPilot بدون نسخ أعمى.

## المطلوب خطوة بخطوة
1. ابدأ git داخل المشروع
2. أنشئ branch باسم week-01-setup
3. أضف الملفات واعمل commit
4. اكتب README صغير يشرح طريقة تشغيل المشروع

## معايير القبول
- المشروع يعمل بدون أخطاء واضحة.
- تستطيع شرح الملفات التي عدلتها.
- يوجد Commit باسم واضح لهذا الدرس.
- النتيجة ظاهرة في المتصفح أو موثقة بصورة/ملاحظة.

## تسليم الواجب
اكتب في نهاية ملف notes.md: ماذا تعلمت؟ وما الخطأ الذي واجهك؟ وكيف حللته؟
