واجب الختام: حفظ التقدم وترتيب الكود

"المبرمج الحقيقي يعرف قيمة Git. اليوم ستقوم بحفظ أول لقطة لمشروعك للأبد."

📂 1. تفعيل المستودع

قم بتنفيذ git init داخل مجلد مشروعك. هل ظهر مجلد مخفي باسم .git؟

📸 2. عمل الـ Commit الأول

أضف جميع الملفات باستخدام git add . ثم احفظ التغييرات برسالة واضحة: git commit -m "feat: setup project environment".

🧹 3. مراجعة النظافة

تأكد من عدم وجود ملفات مؤقتة أو غير ضرورية في مجلد المشروع. راجع أسماء الملفات التي أنشأتها وتأكد أنها باللغة الإنجليزية وتتبع المعايير.

"بإكمال هذا الواجب، تكون قد أتممت تأسيس مشروعك بشكل احترافي عالمي."
