واجب الدرس 3: لمسة الجمال الأولى لـ FlowPilot

"حان الوقت لتجعل FlowPilot يبدو رائعاً! ستستخدم Tailwind CSS لتصميم أول واجهة احترافية."

01 تخصيص ألوان FlowPilot

الهدف:

إضافة ألوان مخصصة لمشروع FlowPilot في ملف الإعدادات.

الخطوات:

  1. 1. افتح ملف tailwind.config.js في VS Code.
  2. 2. ضمن theme.extend.colors، أضف لوناً مخصصاً باسم flowpilot بقيم: 500: '#6366f1' و 700: '#4338ca'.
  3. 3. أضف لوناً ثانياً باسم flowpilot-accent بقيم: 500: '#10b981' (أخضر) و 700: '#059669'.
  4. 4. احفظ الملف.

💡 تذكر: بعد تعديل tailwind.config.js، أعد تشغيل npm run dev ليتم تحديث Tailwind.

ماذا تتحقق منه؟

  • الملف لا يحتوي على أخطاء نحوية (Syntax errors).
  • الألوان الجديدة (flowpilot, flowpilot-accent) ظهرت في VS Code IntelliSense.

03 إضافة 3 بطاقات إحصائيات (Stat Cards)

الهدف:

إنشاء 3 بطاقات إحصائيات باستخدام Grid - مثل لوحة تحكم حقيقية.

الخطوات:

  1. 1. تحت زر "ابدأ رحلتك" مباشرة، أضف هذا الكود:

{/* 3 بطاقات إحصائيات */}

<div className="grid grid-cols-1 md:grid-cols-3 gap-4 mt-10 pt-10 border-t border-slate-100">

  <div className="bg-indigo-50 p-4 rounded-2xl text-center">

    

    

    

  </div>

  <div className="bg-emerald-50 p-4 rounded-2xl text-center">

    

    

    

  </div>

  <div className="bg-amber-50 p-4 rounded-2xl text-center">

</div>

★ التحدي الإضافي (اختياري)

أضف خلفية تفاعلية (Animated Gradient) إلى الصفحة الرئيسية:

  1. أضف كلاس animate-pulse للشعار (الأيقونة) ليجعلها تومض بلطف.
  2. غيّر التدرج اللوني إلى bg-gradient-to-r from-indigo-500 via-purple-500 to-pink-500 للخلفية.
  3. لاحظ كيف يتغير شكل الصفحة بالكامل!

تلميح: كلاس animate-pulse هو كلاس جاهز في Tailwind يجعل العنصر يومض (يظهر ويختفي) ببطء.

04 أسئلة فهم

أجب عن الأسئلة التالية:

  1. ما الفرق بين p-4 و m-4؟
  2. ماذا يعني md:grid-cols-2؟
  3. لماذا نستخدم className بدلاً من class في JSX؟
  4. ما فائدة كلاس hover:bg-*؟

طريقة التسليم

  • • لقطة شاشة 1: المتصفح يعرض بطاقة الترحيب الجديدة.
  • • لقطة شاشة 2: إظهار البطاقات الإحصائية (3 بطاقات) تحت الزر.
  • • لقطة شاشة 3 (اختياري): تأثير animate-pulse أو الخلفية الجديدة.
  • • النص: إجابات الأسئلة النظرية.
  • ⚠️ إذا لم تظهر الألوان flowpilot، أعد تشغيل npm run dev بعد حفظ tailwind.config.js

مخاطر شائعة

  • • نسيان تحديث tailwind.config.js: الألوان المخصصة لن تعمل إذا لم تضفها بشكل صحيح.
  • • عدم إعادة تشغيل npm run dev: بعد تعديل الإعدادات، أعد تشغيل خادم Vite.
  • • خطأ إملائي في الكلاس: Tailwind حساس للحروف. bg-whit لن يعمل، يجب bg-white.
  • • نسيان إغلاق علامات JSX: كل عنصر يجب إغلاقه (self-closing أو closing tag).
