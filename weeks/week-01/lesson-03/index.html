Tailwind CSS وبناء النظام البصري (Design System)

"المظهر الخارجي لا يقل أهمية عن الكود الداخلي. لنجعل FlowPilot يبدو رائعاً منذ البداية."

ما هو Tailwind CSS؟

هل سبق وحاولت تنسيق صفحة HTML وشعرت بالضياع بين ملفات CSS الطويلة؟ Tailwind CSS يحل هذه المشكلة. بدلاً من كتابة ملفات منفصلة، يوفر لك "كلمات جاهزة" (Classes) تضعها مباشرة داخل الـ HTML لتعطيه اللون، الحجم، والمسافات المطلوبة.

مثلاً، بدلاً من كتابة color: blue; font-weight: bold; في ملف خارجي، نكتب فقط text-blue-600 font-bold داخل الكود. هذا يجعل التطوير أسرع بكثير وأكثر تنظيماً.

ما هو "النظام البصري" (Design System)؟

التطبيقات الاحترافية لا تختار الألوان عشوائياً في كل صفحة. بل يحدد المطور "مجموعة ألوان" و "أحجام خطوط" ثابتة يستخدمها في كل مكان. هذا ما نسميه بالـ Design System.

مثال لتقريب الفكرة

تخيل Tailwind كأنه "علبة مكعبات Lego". بدلاً من أن تضطر لصهر البلاستيك وتشكيل المكعب بنفسك (CSS التقليدي)، لديك مكعبات جاهزة بكل الألوان والأشكال. كل ما عليك فعله هو تركيبها بجانب بعضها لتبني قصراً رائعاً!

كيف سنستخدمه في FlowPilot؟

سنستخدم Tailwind لتصميم "واجهة عربية" (RTL) تبدأ من اليمين لليسار. سنحدد ألواننا الرئيسية (النيلي والزمردي) ونبني أول الأزرار والبطاقات التي سيراها مستخدمو FlowPilot.

التطبيق العملي

1

تثبيت Tailwind CSS

داخل مجلد مشروع flowpilot، اكتب الأوامر التالية:

npm install -D tailwindcss postcss autoprefixer

npx tailwindcss init -p

2

تنسيق أول عنصر (Card)

جرب كتابة هذا الكود داخل ملف Welcome.jsx وشاهد كيف يتغير التصميم فوراً:

<div className="bg-white p-8 rounded-3xl shadow-lg border border-slate-200">

  <h1 className="text-2xl font-black text-indigo-600">مرحباً بكم في FlowPilot</h1>

</div>

شرح الكلاسات: bg-white (خلفية بيضاء)، p-8 (مسافة داخلية)، rounded-3xl (حواف دائرية)، shadow-lg (ظل ناعم).
