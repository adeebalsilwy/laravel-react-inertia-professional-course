# واجب الدرس 2: تثبيت Laravel 12 وربطه بـ React وInertia

## الهدف
تطبيق الدرس بنفسك داخل مشروع FlowPilot بدون نسخ أعمى.

## المطلوب خطوة بخطوة
1. أنشئ مشروع Laravel جديد
2. ثبت Starter Kit أو حزم Inertia/React حسب بيئتك
3. شغل npm install ثم npm run dev
4. افتح الصفحة الرئيسية وتأكد من ظهور React

## معايير القبول
- المشروع يعمل بدون أخطاء واضحة.
- تستطيع شرح الملفات التي عدلتها.
- يوجد Commit باسم واضح لهذا الدرس.
- النتيجة ظاهرة في المتصفح أو موثقة بصورة/ملاحظة.

## تسليم الواجب
