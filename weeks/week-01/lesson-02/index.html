تثبيت Laravel 12 وربطه بـ React عبر جسر Inertia.js

"سنقوم اليوم بدمج قوتين عظيمتين: ذكاء Laravel في الخلفية، وجمال وتفاعل React في الواجهة."

لماذا هذا الدرس مهم؟

في الماضي، كان على المبرمج بناء مشروعين منفصلين تماماً (واحد للخلفية وآخر للواجهة). لكن مع Inertia.js، أصبح بإمكاننا بناء تطبيق واحد يجمع بين سهولة Laravel وقوة React.

هذا الدرس سيجعلك قادراً على إنشاء "أول صفحة حقيقية" في تطبيقك، ويفهمك كيف تتدفق البيانات بين السيرفر والمتصفح بسلاسة تامة.

ما هو الـ Stack الذي سنبنيه؟

Inertia.js: الجسر السحري

عادة، عندما تضغط على رابط، يقوم المتصفح بإعادة تحميل الصفحة بالكامل. Inertia يمنع ذلك؛ فهو يجعل تطبيقك يبدو كتطبيق موبايل سريع جداً، مع الحفاظ على سهولة البرمجة بلغة PHP.

لماذا نستخدم React؟

لأنها تسمح لنا بتقسيم الواجهة إلى "قطع صغيرة" (Components). تخيل أن زر "الإعجاب" أو "شريط التنقل" هو قطعة مستقلة يمكنك استخدامها في أي مكان آخر بسهولة.

مثال لتقريب الفكرة

تخيل Laravel كمدير مطعم ذكي يجهز الطلبات، و React كطاولة تقديم أنيقة جداً.

بدون Inertia، سيكون عليك تغيير الطاولة بالكامل في كل مرة يطلب فيها الزبون طبقاً جديداً.

مع Inertia، يقوم النادل (Inertia) بتغيير الأطباق على نفس الطاولة بسرعة فائقة دون إزعاج الزبون!

في مشروع FlowPilot

هذا الدرس هو "شهادة ميلاد" مشروعنا. في نهاية الخطوات العملية، ستمتلك مجلد مشروع FlowPilot الحقيقي الذي سنقضي فيه الثمانية أسابيع القادمة. سنرى أول واجهة لـ FlowPilot تظهر في المتصفح!

التطبيق العملي

1

إنشاء مشروع Laravel جديد

افتح الـ Terminal وانتقل للمجلد الذي تضع فيه مشاريعك، ثم اكتب:

composer create-project laravel/laravel flowpilot

cd flowpilot

2

تثبيت Inertia.js (جانب السيرفر)

نحتاج لإضافة مكتبة Inertia داخل Laravel ليفهم كيف يتخاطب مع React.

composer require inertiajs/inertia-laravel

3

تثبيت React (جانب الواجهة)

الآن نثبت مكتبة React والأدوات المرتبطة بها باستخدام NPM.

npm install @inertiajs/react react react-dom @vitejs/plugin-react

شرح الكود

return Inertia::render('Welcome');

هذا السطر (الذي يوضع في Controller) هو أهم سطر اليوم!
- Inertia::render يخبر Laravel: "لا تعرض صفحة عادية، بل شغل محرك React".
- 'Welcome' هو اسم الملف الذي أنشأناه في مجلد React.
