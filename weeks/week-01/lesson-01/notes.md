# ملاحظات الدرس 1: تهيئة بيئة العمل وفهم خريطة المشروع

## ملخص سريع
نثبت الأدوات المطلوبة ونفهم لماذا نستخدم Laravel وReact وInertia معاً قبل كتابة أول كود.

## أهم المصطلحات
- ما هو Laravel وما دوره في الخلفية Backend
- ما هو React وما دوره في الواجهة Frontend
- ما هو Inertia ولماذا يربط الطرفين بدون API منفصل
- معنى Composer وNode وNPM وVite وGit
