# واجب الدرس 1: تهيئة بيئة العمل وفهم خريطة المشروع

## الهدف
تطبيق الدرس بنفسك داخل مشروع FlowPilot بدون نسخ أعمى.

## المطلوب خطوة بخطوة
1. افتح Terminal وتأكد من وجود PHP وComposer وNode
2. أنشئ مجلد عمل باسم flowpilot-course
3. جهز محرر VS Code وافتح المجلد
4. اكتب ملف ملاحظات يشرح دور كل تقنية بجملة واحدة

## معايير القبول
- المشروع يعمل بدون أخطاء واضحة.
- تستطيع شرح الملفات التي عدلتها.
- يوجد Commit باسم واضح لهذا الدرس.
- النتيجة ظاهرة في المتصفح أو موثقة بصورة/ملاحظة.

## تسليم الواجب
اكتب في نهاية ملف notes.md: ماذا تعلمت؟ وما الخطأ الذي واجهك؟ وكيف حللته؟
