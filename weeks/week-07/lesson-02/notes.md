# ملخص الدرس: React Component Quality

## الفكرة الأساسية
جودة المكون تعني أن يكون صغيراً، واضح الاسم، يستقبل props محددة، ولا يحمل مسؤوليات كثيرة في مكان واحد.

## لماذا هذا مهم؟
واجهة React قد تتحول بسرعة إلى فوضى إذا كانت الصفحة الواحدة تحتوي كل شيء. الجودة تجعل التطوير أسرع والأخطاء أقل.

## داخل FlowPilot
بطاقات المشاريع والمهام والنماذج ستصبح مكونات نظيفة يمكن إعادة استخدامها.

## المثال
function TaskBadge({ status }) {
  const labels = { todo: "لم تبدأ", doing: "قيد التنفيذ", done: "مكتملة" };
  return <span className="rounded-full bg-slate-100 px-3 py-1">{labels[status]}</span>;
}

## نتيجة الدرس
مكونات React أنظف وأسهل في الصيانة.
