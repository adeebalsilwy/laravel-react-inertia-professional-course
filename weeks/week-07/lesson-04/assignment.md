# واجب الدرس 28: Security Hardening

## الهدف
تطبيق الدرس بنفسك داخل مشروع FlowPilot بدون نسخ أعمى.

## المطلوب خطوة بخطوة
1. راجع كل requests
2. تأكد من fillable
3. تأكد من Policies
4. لا ترفع .env إلى Git
5. اختبر مستخدماً غير مالك

## معايير القبول

- تستطيع شرح الملفات التي عدلتها.
- يوجد Commit باسم واضح لهذا الدرس.
- النتيجة ظاهرة في المتصفح أو موثقة بصورة/ملاحظة.
