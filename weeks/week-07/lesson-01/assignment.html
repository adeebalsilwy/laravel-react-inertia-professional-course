✍️ واجب: كتابة اختبارات لمهام FlowPilot

في هذا الواجب، ستطبق بنفسك ما تعلمته عن الاختبارات. ستكتب 4 اختبارات Feature تغطي حالات مختلفة في مشروع FlowPilot.

📝 المهمة 1: إنشاء ملف اختبار المهام

افتح التيرمنال في مجلد مشروع FlowPilot وشغل الأمر التالي:

php artisan make:test TaskTest

تأكد من ظهور رسالة Test created successfully. الآن افتح الملف tests/Feature/TaskTest.php وستجد هيكلاً جاهزاً للكتابة.

📝 المهمة 2: اختبار عرض صفحة المهام

اكتب أول دالة اختبار داخل TaskTest تتأكد من أن صفحة المهام تعرض بشكل صحيح:

public function test_tasks_page_loads(): void

{

    $user = User::factory()->create();

    $this->actingAs($user);

    $response = $this->get('/tasks');

    $response->assertStatus(200);

}

ملاحظة: استخدم actingAs لأن صفحة المهام محمية وتتطلب تسجيل دخول.

📝 المهمة 3: اختبار منع إنشاء مهمة بدون عنوان

القاعدة: حقل title إجباري. اكتب اختباراً يتأكد من رفض الطلب إذا كان العنوان فارغاً:

public function test_task_requires_title(): void

{

    $user = User::factory()->create();

    $this->actingAs($user);

    $response = $this->post('/tasks', [

        'title' => '',

        'description' => 'وصف',

    ]);

    $response->assertSessionHasErrors('title');

}

شرح: assertSessionHasErrors('title') تتأكد من ظهور رسالة خطأ للحقل title. هذا يعني أن التحقق (Validation) يعمل بشكل صحيح.

📝 المهمة 4: اختبار حذف مهمة

اكتب اختباراً يتأكد من أن المستخدم يمكنه حذف مهمته الخاصة:

public function test_user_can_delete_own_task(): void

{

    $user = User::factory()->create();

    $this->actingAs($user);

    $task = Task::factory()->create([

        'user_id' => $user->id

    ]);

    $response = $this->delete("/tasks/{$task->id}");

    $response->assertStatus(302);

    $this->assertDatabaseMissing('tasks', [

        'id' => $task->id

    ]);

}

شرح: assertDatabaseMissing() تتأكد أن المهمة حُذفت من قاعدة البيانات ولم تعد موجودة.

⭐ المهمة الإضافية (تحدي): اختبار الصلاحيات

اكتب اختباراً يتأكد من أن المستخدم "لا يستطيع" حذف مهمة ليست ملكه:

public function test_user_cannot_delete_others_task(): void

{

    $owner = User::factory()->create();

    $intruder = User::factory()->create();

    $task = Task::factory()->create([

        'user_id' => $owner->id

    ]);

    $this->actingAs($intruder);

    $response = $this->delete("/tasks/{$task->id}");

    $response->assertStatus(403);

}

شرح: هنا ننشئ مستخدمين: صاحب المهمة ومستخدم آخر. نتأكد من رفض الحذف برمز 403.

🏁 الخطوة النهائية

شغل الأمر التالي في التيرمنال:

php artisan test --filter TaskTest

إذا رأيت كل الاختبارات خضراء ✅، فهنيئاً! أنت الآن تفهم كيف تكتب اختبارات تحمي تطبيقك.
