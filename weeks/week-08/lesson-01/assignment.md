# واجب الدرس 29: Production Build وEnvironment

## الهدف
تطبيق الدرس بنفسك داخل مشروع FlowPilot بدون نسخ أعمى.

## المطلوب خطوة بخطوة
1. انسخ .env.example
2. اضبط APP_ENV وAPP_DEBUG
3. شغل build
4. تأكد أن assets تعمل

## معايير القبول
- المشروع يعمل بدون أخطاء واضحة.
- تستطيع شرح الملفات التي عدلتها.
- يوجد Commit باسم واضح لهذا الدرس.
- النتيجة ظاهرة في المتصفح أو موثقة بصورة/ملاحظة.

## تسليم الواجب
