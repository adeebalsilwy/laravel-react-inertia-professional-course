ما هو الـ Production Build؟

طوال الدورة، كنت تشغل npm run dev. هذا الأمر يجعل التطبيق مرناً للتعديل، لكنه ليس سريعاً بما يكفي للمستخدمين الحقيقيين. الـ Build هو عملية "ضغط وتقليص" ملفات React و Tailwind لتصبح ملفاً واحداً صغيراً جداً يفتح في لمح البصر.

التطبيق العملي

1

إنتاج ملفات الواجهة النهائية

اكتب الأمر التالي لمرة واحدة فقط قبل رفع المشروع:

npm run build

بعد هذا الأمر، سيقوم Vite بإنشاء مجلد public/build الذي يحتوي على ملفاتك الجاهزة للعمل.
