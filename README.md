# FlowPilot Academy - النسخة المحسنة خطوة بخطوة

مشروع HTML وJavaScript وCSS لعرض دورة تدريبية عربية لمدة شهرين حول Laravel 12 وReact وInertia.js وTailwind CSS.

## محتويات النسخة

- 8 أسابيع تدريبية.
- 32 درساً.
- كل درس يحتوي على شرح نظري وعملي خطوة بخطوة.
- صفحة واجب احترافية لكل درس.
- صفحة كويز تفاعلية لكل درس.
- مخططات SVG وصور تعليمية.
- صفحة خطة تطوير للمشروع.
- بحث وفلترة داخل الدروس.

## طريقة التشغيل

افتح ملف `index.html` مباشرة في المتصفح.


## التحسين الأخير

تمت إضافة صفحة مراجع تعليمية، وتطوير خطة الدروس اعتماداً على موضوعات شائعة في دورات Laravel العربية والتوثيق العربي، مع إعادة صياغة الشروحات بالكامل لتناسب FlowPilot Academy. لا يوجد نسخ حرفي من المصادر؛ تم استخدام الموضوعات كإلهام لبناء خطة تعليمية مرتبة.

## الملفات الجديدة المهمة

- pages/references.html
- pages/plan.html
- تحسينات داخل صفحات الدروس تشمل قسم: كيف تم تطوير موضوع هذا الدرس؟
- تحسينات داخل صفحات الدروس تشمل قسم: خطة التطبيق العملية المطورة لهذا الدرس


## تحديث الاقتباس بالمعنى من المراجع

تمت إضافة صفحة مراجع مطورة وخطة تدريبية أكثر وضوحاً، مع تضمين موضوعات مثل متطلبات Laravel، Composer، public، config، .env، key، خادوم التطوير المحلي، MVC، Routing، Controllers، Eloquent، Validation، Authentication، Authorization، Testing، Deployment.

المحتوى لا ينسخ نصوص المراجع حرفياً، بل يعيد صياغة الموضوعات بأسلوب تعليمي أصلي موجه لطلاب FlowPilot Academy.
