# FlowPilot Academy - Laravel 12 + React + Inertia.js

هذه نسخة تعليمية جاهزة للعرض على الطلاب. تم تعديل الدروس بحيث تكون الشروحات موجهة للطلاب مباشرة، وليست تعليمات لصاحب المشروع أو المدرب.

## ما الذي تحتويه النسخة؟

- 8 أسابيع تدريبية.
- 32 درساً منظماً.
- كل درس داخل مجلد مستقل داخل أسبوعه.
- كل درس يحتوي على:
  - index.html شرح نظري وعملي موجه للطلاب.
  - assignment.html صفحة واجب احترافية.
  - quiz.html صفحة كويز احترافية.
  - diagram.svg مخطط توضيحي.
  - visual-example.svg صورة مبسطة.
  - notes.md ملخص الدرس.
  - assignment.md نسخة نصية من الواجب.
  - quiz.json بيانات الكويز.

## أسلوب الشرح

كل درس يشرح: ما هو المفهوم، لماذا يستخدم، كيف يعمل، أين يظهر داخل FlowPilot، ثم يقدم مثالاً وتطبيقاً عملياً.
