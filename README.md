# FlowPilot Academy - Laravel 12 + React + Inertia.js للمبتدئين

هذه نسخة تعليمية منظمة للمبتدئين تماماً.

## ماذا تحتوي؟
- 8 أسابيع تدريبية مكثفة.
- 32 درساً عملياً خطوة بخطوة.
- كل درس داخل مجلد مستقل ضمن أسبوعه.
- لكل درس: index.html، diagram.svg، assignment.md، quiz.json، notes.md.
- شروحات عربية واضحة تبدأ من الصفر وتشرح لماذا نكتب كل خطوة.
- تنقل سريع وبحث داخل صفحات الدروس والأسابيع.

## طريقة التشغيل
افتح ملف `index.html` في المتصفح.

## هيكل الدروس
`weeks/week-01/lesson-01/index.html` وهكذا حتى `weeks/week-08/lesson-04/index.html`.

## المشروع العملي
كل الدروس مرتبطة بمشروع واحد اسمه FlowPilot، وهو منصة لإدارة المشاريع والمهام باستخدام Laravel وReact وInertia.
